    • About CocoaCare

    CocoaCare is a web-based diagnostic tool designed to support cocoa farmers by enabling early detection of common leaf diseases using artificial intelligence. This application was developed as part of a final year project in Computer Engineering, Covenant University. This project combines computer vision, responsive web design to provide real-world agricultural support.

    Farmers can either upload or take a photo of a cocoa leaf, and the app will analyze the image using a trained deep learning model to identify possible diseases. It then returns:

    • A predicted disease name
    • A confidence score indicating the model‚Äôs certainty
    • Clear, practical recommendations tailored to the diagnosis

    CocoaCare is built for ease of use on both mobile and desktop devices, It features like theme switching (light/dark mode) and upload history.

    By offering farmers quick, insights into plant health, this research project aims to promote informed decision-making and help reduce crop loss through timely intervention.
